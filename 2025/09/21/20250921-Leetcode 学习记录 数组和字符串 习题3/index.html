其实这道题挺难的，至少我消耗了两天。

题目是合并区间，实际上给定了一组区间，找出其中有重叠的区间，合并。合并后返回所有没有重叠的区间。
链接：https://leetcode.cn/leetbook/read/array-and-string/c5tv3/

这道题给的示例都是有序数组哦，所以我写代码的时候就是假设数组有序，很坑，只通过三分之一左右的测试用例。Leetcode很好的一点是，没有通过的测试用例会告知你哦，所以你就可以看到哪些例子没有通过。

我当然想找数组排序的函数啦，但是官方手册里也没有翻到，所以我就看了别人的题解。恩，果然有收获。

因为这几天学会了一点HashMap的知识，当然就要新鲜的用起来，所以代码就是用HashMap实现的结果收集。

class Solution {
    public int[][] merge(int[][] intervals) {

        if( intervals.length <= 1){
            return intervals;
        }

        Arrays.sort(intervals, Comparator.comparingInt(a -> a[0]));//按每行的第0列升序排序(这一行是抄别人的哈）

        int start, end;

        start = intervals[0][0];
        end = intervals[0][1];

        Map<Integer, Integer> nums = new HashMap<>(); 

        for( int i = 1; i < intervals.length; i ++){

      
            if( intervals[i][0] > end || intervals[i][1] < start){
                nums.put(start,end);
                start = intervals[i][0];
                end = intervals[i][1];
            }
            
            if( intervals[i][1] >= start && intervals[i][1] > end ){
                end = intervals[i][1];
                
            }
            if( intervals[i][0] <= end && intervals[i][0] < start ){
                start = intervals[i][0];
            }
        }

        if( nums.containsKey(start) == false){
            nums.put( start, end);
        }

        int[][] result =  new int[ nums.size()][2];

        Set<Map.Entry<Integer, Integer>> entries = nums.entrySet();
        
        int j = 0;
        for(Map.Entry<Integer,Integer> entry : entries){
            result[j][0] = entry.getKey();
            result[j][1] = entry.getValue();
            j++;
        }
        return result;    

    }
}


时间上复杂度有些高， 因为HashMap转成数组又加了一个循环。我看到大家大多使用List收集的结果，然后List.toArray()就好了。

还是不熟悉各种函数啊，我得彻底补习一下！
