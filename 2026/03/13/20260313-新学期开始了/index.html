关于Python

实话说，不怎么喜欢它，但是确实很有用，于是今年开始会用到很多。发现Python的有些函数还是很好用，mark一下。

因为要查询季度数据，但又要用日期作为参数，所以需要提取每个季度最后一天的日期，PeriodIndex真的有帮很大忙，这段代码发这里，有需要的同志们可以自取。

from datetime import timedelta

date = pd.date_range(start = '2020-01-01',end = '2022-01-01',freq= 'D')
# 生成季度标签
quarter = pd.PeriodIndex(date, freq='Q', name='Quarter')
# 将周期序列转换成日期序列
quarter = quarter.to_timestamp()
